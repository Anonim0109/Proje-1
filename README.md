# Proje-1

PRATİK EĞLENCE

BU UYGULAMA BASİT EĞLENCE VE KOLAYLIK AMAÇLIDIR

-Eğer anasayfada 1.ye basılırsa:
hesaplamalar açılır ve kolayca aklınızdan yapamadığınız 4 işlemi burada yapabilirsiniz.

-Eğer anasayfada 2.ye basılırsa:
istediğiniz şekli karşısındaki numarayı yazarak çizdirebilirsiniz.

-Eğer anasayfada 3.ye basılırsa:
taş, kağıt, makas oyununu yada kısa bir macera oyununu seçip oynayabilirsiniz.

-Eğer anasayfada 4.ye basılırsa:
makine ile ufak çaplı bir sohbet edebilirsiniz.

-Eğer anasayfada 5.ye basılırsa:
makine sizi ufak bir genel kültür soruları sorarak sizi ölçer.

-Eğer anasayfada 6.ya basılırsa:
not ortalamalarınızı yazarak alacağınız belgeyi makine size söyler.

-Eğer anasayfada 7.ye basılırsa:
çarpım tablosunu sizin önünüze döker ve kolayca bakıp işlerinizden süre kazanabilirsiniz

-Eğer anasayfada 8.ye basılırsa:
uygulamadan kolayca çıkabilirsiniz.
